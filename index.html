Course Planner - Enhancements Overview

Welcome! This page showcases the Course Planner project and its three major enhancements, demonstrating improvements in functionality, efficiency, and user interaction.

Features:

  • Load course data from a CSV file into a BST
  • Search for courses by ID in BST
  • Sort courses by the number of prerequisites in BST
  • Interactive keyword-based search in BST (now case-insensitive!)
  • Load courses from MongoDB
  • Search for a course in MongoDB
  • Graphic User Interface (GUI) using Tkinter for improved usability

  • Handled file I/O operations using Pythons built-in csv module.
  • Refactored C++ memory management into Pythons garbage collection system.

üì• Installation & Usage

To install and run the Course Planner with MongoDB integration, follow these steps:

Installation & Setup:

  1. Clone the Repository: 
    git clone https://github.com/bpnavy03/bpnavy03.github.io.git
cd bpnavy03.github.io
  2. Install Dependencies: 
    pip install pymongo
    pip install tk
  3. Prepare the course data: 
    The program can read course data from a CSV file. This file has been provided.
    File Name: courses.csv
    Location: courses.csv must be in the same directory as course_planner.py
  4. Setup MongoDB (optional): 
    Install/verify installation of MongoDB
  5. Import Course Data into MongoDB: 
    Start MongoDB: mongod --dbpath "C:\Program Files\MongoDB\Server\8.0\data"
    Open the Mongo Shell: mongosh
    Switch to the course database: use course_planner
    Import the CSV file into MongoDB using:
    mongoimport --db course_planner --collection courses --type csv --file courses.csv --headerline
    Verify CSV file was imported into MongoDB correctly: db.courses.find().pretty()

Usage Instructions:

  1. Running in GUI mode: 
    Open a command prompt/windows PowerShell/terminal, navigate to the program directory, and run: python course_planner.py
    The Tkinter GUI will launch, providing buttons for loading course data, searching, and sorting courses.
    Start with option 1 and enter courses.csv. The CSV file should be loaded successfully.
    Then use option 6 to load course data from MongoDB.
    The GUI includes interactive elements such as dropdowns, text input, and buttons for a seamless experience.
  2. Running the Command-Line Interface (CLI) Mode 
    Open a command prompt/windows PowerShell/terminal and navigate to the program directory.
    Run: python course_planner.py
    Navigate the menu using the provided options.
    Start with option 1 and enter courses.csv. The CSV file should be loaded successfully.
    The use option 6 to load course data from MongoDB.
    Use options 2, 3, 4, or 5 to perform course searches. Use option 9 to exit the program.
