Enhancements Summary

Enhancement 1: C++ to Python Conversion

The initial version of the Course Planner was implemented in C++. The first enhancement involved converting the program to Python, ensuring functional parity while leveraging Pythons simplicity and extensive libraries.

  • Converted Binary Search Tree (BST) implementation to Python.
  • Handled file I/O operations using Pythons built-in csv module.
  • Refactored C++ memory management into Pythons garbage collection system.

Enhancement 2: Sorting & Interactive Search

The second enhancement introduced two major features: sorting courses by the number of prerequisites and an interactive search function.

  • Implemented a sorting algorithm to organize courses based on prerequisite count.
  • Developed an interactive keyword-based search, allowing users to find courses dynamically.
  • Improved user experience with enhanced error handling and formatted output.

Enhancement 3: Database Integration Using MongoDB

The third enhancement replaces the previous BST-only structure with a MongoDB database, providing persistent storage and better scalability.

  • Stored course data in MongoDB, allowing data to persist beyond program execution.
  • Developed MongoDB queries for course retrieval and searching.
  • Enabled dual functionality, allowing users to choose between CSV or DB-based operations.

Finalized Enhanced Artifact with GUI

Introduced a Tkinter-based GUI for an improved user experience.

  • Designed a main window with buttons to load, search, and view courses.
  • Added input fields for user-friendly course searching.
  • Implemented list displays for course data from CSV and MongoDB.
  • Enhanced application accessibility by removing the need for command-line interaction.

📥 Installation & Usage

To install and run the Course Planner with MongoDB integration, follow these steps:

  1. Clone the Repository: 
    git clone https://github.com/bpnavy03/bpnavy03.github.io.git
cd bpnavy03.github.io
  2. Install Dependencies: 
    pip install pymongo
    pip install tk
  3. Run the Program: 
    python course_planner.py
